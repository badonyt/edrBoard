Create input

To create your own input youre going to create a json

Which inside youre going to write { "W": { "actual": "w", "x": "150px", "y": "100px", "width": "250px", "height": "100px" }, }

You might not understand what is going, which is why ill explain now.

Understanding what it means

So first the "W": is what is going to appear in the view

The x and the y are where its going to appear in the screen. USE PX(if you know what you are doing make sure to use css)

The width and the height dictate how big the box is. USE PX(if you know what you are doing make sure to use css)

Finally the "actual" is the identifier for the key. To check what you have to put in there check this guide here
